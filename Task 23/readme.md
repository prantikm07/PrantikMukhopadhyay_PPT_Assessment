# Task 23: Azure Resource Manager & Cost Management (Easy)

## Simple Definition

Azure Resource Manager (ARM) organizes resources into resource groups. Cost Management helps track and control cloud spending.

## Objective

To create a resource group and monitor cost analysis in Azure.

## Architecture Overview

Subscription  
↓  
Resource Group  
↓  
Resources  
↓  
Cost Management Dashboard

## Steps

### Step 1: Create Resource Group

1. Go to Azure Portal.
2. Click Resource Groups → Create.
3. Enter group name.
4. Select region.
5. Create resource group.

### Step 2: View Cost Analysis

1. Go to Cost Management + Billing.
2. Select subscription.
3. Click Cost Analysis.
4. View spending breakdown.

### Step 3: Set Budget (Optional)

1. Go to Budgets.
2. Create budget.
3. Set threshold alert.

## Verification

* Resource group visible.
* Cost analysis dashboard visible.
* Budget alert configured (if created).

## Conclusion

Successfully created resource group and explored Azure cost management features, demonstrating resource organization and budget monitoring.
