# Task 19: Azure Storage Account (Blob, File, Table)

## Simple Definition

Azure Storage Account provides cloud storage services including Blob Storage (object storage), File Storage (managed file shares), and Table Storage (NoSQL structured storage).

## Objective

To create an Azure Storage Account and explore Blob, File, and Table storage services.

## Architecture Overview

User  
↓  
Azure Storage Account  
↓  
Blob Storage | File Share | Table Storage

## Steps

### Step 1: Create Storage Account

1. Go to Azure Portal.
2. Search for Storage Accounts → Create.
3. Select Subscription and Resource Group.
4. Enter unique storage account name.
5. Select Region.
6. Choose Standard performance.
7. Create storage account.

### Step 2: Upload File to Blob Storage

1. Open Storage Account.
2. Go to Containers → Create container.
3. Upload a sample file.
4. Verify file upload.

### Step 3: Create File Share

1. Go to File shares.
2. Click Create file share.
3. Set quota.
4. Upload a test file.

### Step 4: Create Table

1. Go to Tables.
2. Create new table.
3. Insert sample entity.

## Verification

* Storage account created successfully.
* Blob container shows uploaded file.
* File share created and accessible.
* Table created and entity inserted.

## Conclusion

Successfully created Azure Storage Account and explored Blob, File, and Table storage services, demonstrating multi-storage capabilities in Azure.
