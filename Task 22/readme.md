# Task 22: Azure Monitoring & Diagnostics (Easy)

## Simple Definition

Azure Monitor helps track performance, availability, and diagnostics of Azure resources.

## Objective

To enable monitoring and view performance metrics for Azure resources.

## Architecture Overview

Azure Resource  
↓  
Azure Monitor  
↓  
Metrics & Logs

## Steps

### Step 1: Enable Monitoring

1. Go to Azure Portal.
2. Open Virtual Machine.
3. Click Monitoring → Insights.
4. Enable monitoring.

### Step 2: View Metrics

1. Go to Monitor → Metrics.
2. Select resource.
3. Choose metric (CPU usage).
4. View graph.

### Step 3: View Logs

1. Go to Logs.
2. Run basic query to retrieve resource logs.

## Verification

* Metrics graph visible.
* CPU utilization displayed.
* Logs accessible.

## Conclusion

Successfully enabled Azure monitoring and viewed diagnostic metrics, demonstrating performance tracking and analysis.
